## Image Requirements

### File Specifications
- **Format**: JPG (preferred) or PNG
- **Quality**: High resolution, professional
- **Aspect Ratio**: 4:3 or 16:9 landscape
- **File Size**: 50-200KB (web-optimized)
- **Dimensions**: Min 800px width

### File Naming (CRITICAL)
**Must match exactly** (case-sensitive):
- `bullet-camera.jpg`
- `dome-camera.jpg`
- `ptz-camera.jpg`
- `lpr-camera.jpg`

### Content Guidelines
✅ Professional photos of actual cameras

✅ Good lighting

❌ Stock illustrations

### Troubleshooting
**Images not showing?**
- Check file names match exactly
- Verify files are in correct folder
- Clear browser cache
- Check file format (JPG/PNG)

**Text not readable?**
- Image may be too dark/light
- Try different composition
- Ensure good contrast

**Zoom looks bad?**
- Check image resolution (min 800px)
- Verify aspect ratio (4:3 or 16:9)
- Ensure image quality is high
