## Image Guidelines

- **Format**: JPG (recommended) or PNG
- **Quality**: High resolution (at least 800px width)
- **Aspect Ratio**: 16:9 or similar landscape format
- **File Size**: Optimize for web (50-200KB recommended)
- **Content**: Professional photos showing the actual services/equipment

## Technical Implementation

Images are automatically optimized by Next.js with:
- Lazy loading for performance
- Responsive sizing for mobile, tablet, and desktop
- Smooth zoom effect on hover
- SEO-friendly alt text
